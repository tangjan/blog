个人小站开张 🎉

更新于2023-10-04|未分类
|字数总计:2.3k|阅读量:|评论数:

序

缘起

最开始接触并建设个人网站是在 2021 年 11 月的大四上学期，距今已跨越将近两年的时光了。

这两年间本科毕了业、首次参加了工作，生活时平淡时起伏。

日月星辰就在这样的生活中不断地默默交替着，回过神来发现自己已经第一次辞了职。在敲下这段文字的上个瞬间，才朦胧地意识到自己已经成为了本科期间还不理解的「大人」。

生活日复一日地推进，将我带入一个又一个波流之中。

有时，我身心疲惫地怀疑工作与人生的意义，浮躁地寻找工作与生活的平衡点，渴求能承载自己浮浮沉沉的精神归宿；

有时，我羡慕他人的积累与成就，否定自己以往的人生，试图在人生轨迹上另辟蹊径；

有时，我思考「人是社会关系的总和」的含义，渴求与人建立连接，寻求名为「自我」的身份定义；

……

琐碎的信息与事务穿插到我三点一线的日常，迷雾般将我的思绪笼罩。

破局

在一个又一个谜题面前，「个人网站」像万能公式一样，为我提供一个又一个答案。

精神归宿 —— 打开网页浏览器，输入自己的域名，打开就是了。

另辟蹊径 —— 在个人网站上，一切想法与成果皆可化为文字与影像。

身份定义 —— 将自己的存在量化为一个个网站、一篇篇博客，藉此定义「糖加盐」这个存在。

……

如果说「互联网」为我点亮了通往出路的灯塔，那么「个人网站」便为我铺就了坚实的道路，让我能一步步脚踏实地地前进。

规划

目前可以确定的个人站点有：

  • www.tangjiayan.cn / tangjiayan.cn：个人网站导航，展示个人站点列表和 SNS 链接。
  • notes.tangjiayan.cn：个人笔记站，可以公开的学习笔记都会记在此站。生命不息，学习不止，此站很长一段时间内都将是我的网站中更新频率最高的一个，所以它也被我映射为了 GitHub Pages 主域名 tangjan.github.io。
  • blog.tangjiayan.cn：个人博客，即本站，我将用其记录个人的 生活/技术/写作 等。虽然更新频率不如笔记站高，但本站的内容才是我建设网站的初衷。内容不定期更新，但一定是持续更新。我甚至预想到我年老时会发表「玫瑰海棠的正确养护方式」系列博客……
  • cv.tangjiayan.cn：个人在线简历。至少直到财富自由之前，这个站点都大有用处。

这些站点是我目前的想法，持续更新的站点列表会展示在 关于 页面。

建站记

其他博客平台

一开始我是在 博客园、CSDN 上写博客，但由于它们有审核机制、个性化功能不足、有广告等缺点，让我很难静下心来写博客输出。于是调查一番，有了建设个人网站的想法。

服务器

在 2021 年 12 月在腾讯云趁新人特惠花 ￥150 买了三年的腾讯云轻量应用服务器，用 WordPress 框架搭建了第一个个人网站。

考虑国内的服务器需要备案，比较麻烦，于是在今年的 3 月又在海外云服务商 RackNerd 花 $25.46 买了两年的轻量级服务器。

后来出于稳定性、持久性和维护方便性等考虑，决定不再使用服务器自建 WordPress 站点，而是依托于 GitHub Pages 和静态网站生成器（参见 Static Site Generators）进行建站。

暂时不太有兴趣自己开发维护一套主题，所以使用哪个静态网站框架取决于哪里有我喜欢的开源主题，目前在使用的是 VitePress 和 Hexo。

用不上 WordPress，也就用不上服务器了，所以上述买的两台服务器目前处于闲置状态，有机会再考虑利用起来做些好玩的项目。

域名

最开始是在阿里云买的域名 tangjiayan.com，将 www.tangjiayan.com 部署为上文提到的腾讯云轻量应用服务器上的 WordPress 博客。由于是国内的服务器，还历经半个多月的时间进行了备案。

由于那时不了解配置二级域名，于是又买了个 tangjiayan.cn，将 www.tangjiayan.cn 部署为我的旧笔记站——糖加盐的笔记站。

目前主要使用 .cn 的域名来部署个人网站，.com 的域名目前只有 cdn.tangjiayan.com 用于七牛云的图床，以后大概会有派上用场的地方。

这两个域名我都打算一直续费持有，反正也不贵。

图床

一开始是用的 jsdelivr + GitHub 作为图床，用 PicGo 进行管理，但在中国由于网络审查的原因，时不时会出现图裂的问题。

后来也尝试过 SM.MS、路过图床 等免费图床，但这些图床有容量限制，并且我希望能统一管理自己的图片，所以也没有作为主力图床使用。

后来了解到 七牛云的对象存储Kodo 有 10G 的标准存储免费空间 和 每个月10G 的免费标准存储 CDN 回源流出流量，就把七牛云作为了主力图床。

免费额度须知 - 七牛开发者中心

之后会考虑再接入又拍云，因为又拍云也有免费的额度。

又拍云联盟 - 开发者帮助计划，为开发者加速 - 又拍云

根据网站受众的不同，我也会在一些站点继续使用 GitHub 作为图床。

成本

维护个人网站的成本主要分为 时间 和 经济支出 两个方面。

时间

我认为建设个人网站的过程中最耗时间的阶段是起步的时候，具体包括：

  • 对网站的基本认知（服务器、客户端、域名解析 等）
  • 了解建设网站所需用到的技术栈
  • 个人网站的内容及域名规划
  • 网站框架、主题的对比选取

而这个最耗时间的阶段我已度过，对于网站的维护目前还算得心应手，所以不会太多浪费时间在无用功上，时间消磨得还算愉快，不会有太多的沉没成本。

另外，让我愿意付出时间的一个比较重要的原因是：通过建设网站我可以学到很多技术、接触很多信息。

所以我每天哪怕再忙再累，也会愿意抽出至少半个小时的时间维护个人网站。

总之，时间成本不足挂齿。

经济支出

域名支出

域名支出是主要的支出，tangjiayan.com 大概 ￥80/年，tangjiayan.cn 大概 ￥40/年，加一起也就一年一百多块钱，不成问题。

服务器支出

因为主要将站点托管在 GitHub Pages 上，需要使用云计算的评论系统 Waline 短期内使用免费的 LeanCloud 开发版也足够用了，所以在可预见的未来，服务器的支出几乎没有。

图床支出

使用七牛云、又拍云的免费额度再加上 GitHub、免费图床，够用了，所以图床的支出也几乎没有。

当然如果图床受到了 DDOS 攻击另谈……

踏上旅程

一边回忆自己这两年的建站历程，一边撰写这篇博客，断断续续累计花了五六个小时的时间。

之所以愿意投入这么多时间在这篇博客上，是因为经过两年间零零碎碎的积累，我清晰地感受到我对个人网站取得了从0到1的质变性认知，这篇博客便是从0到1的阶段性总结，颇具纪念意义。

本站的开放也了结了我心中的一桩大事 —— 释放表现欲的渠道。
现实中的我不善言辞，稍偏内向，时常苦于没有合适的机会表现自己，缺乏自信和自我认同感。
而于我而言，个人网站便是释放表达欲的绝佳载体，伴随着她们的存在，我将能以强大的心态踏上人生旅程。

不论今后生活如何浮沉，互联网和个人网站都将伴随我度过每一个繁星满天的夜晚。
我将在人类的信息海洋中汲取真谛，成为一个独立思辨的个体。

至少在我财富自由之前，我决定将人生以个人网站为主线进行展开。
反正人生的意义是自己定的，个人网站就是我定下的人生意义之一。

我还记得建设个人网站的那段时间，聆听着触动心弦的歌曲，心脏咚咚地雀跃高歌。

骑车划破强风，无比自由地穿梭于街道灯光之中，在脑海中构筑起人生的蓝图。
